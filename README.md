# Personal Compile Tools

## Origin

This repo was created to port existing code in my "Personal Image Viewer" into its own package. I wanted to expand the scope of the code, make it more generic, and reach 100% test coverage.

## What does this package do

Provides many tools that simplify common workflows when compiling a python module (with something like nuitka). These include file operations, version manipulations, and parsers for requirements files.
